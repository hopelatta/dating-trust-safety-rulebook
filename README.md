# Dating Trust & Safety Rulebook

This repository contains working notes on trust & safety challenges specific to
relationship-driven platforms (e.g. dating, matchmaking, and intimacy-oriented services).

The focus is not content moderation alone, but the decisions that shape user behavior.

These notes are exploratory and meant to be part of a broader understanding rather than a prescribed one-size-fits-all solutions. 

## Scope

Topics covered include:
- Threat modeling for relationship platforms
- Reporting and response workflows
- Ban evasion
- Repeat-offender dynamics
- Identity-verification
